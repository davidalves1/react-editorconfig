# react-editorconfig
A simple cli to create the .editorconfig of the your react project

## Getting Started

### Install
NPM:  
`npm install -g react-editorconfig`

Yarn:  
`yarn global add react-editorconfig`

### Use
In your project path just execute:
`redit`
